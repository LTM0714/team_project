# 🧭 마애리아 (MaAREA)

> **사진 한 장으로, 그곳의 이야기를 기록하다.**  
> 마애리아는 사진 속 지역을 자동으로 인식해 지도 위에 표시해주는 위치 기반 여행 SNS 앱입니다.  
> 사용자는 사진을 업로드하면, 앱이 자동으로 GPS 메타데이터를 분석해 해당 장소를 지도에 표시합니다.  
> 메타데이터가 없을 경우에는 사용자가 직접 지역을 선택할 수 있으며,  
> 선택된 위치는 지도 마커로 시각화되어 여행의 발자취를 한눈에 볼 수 있습니다.

---

##  주요 기능

###  1. 위치 자동 인식
- 업로드한 사진의 **EXIF 메타데이터(GPS 정보)** 를 분석하여 자동으로 위치를 추출
- 지도 위에 **마커(Marker)** 로 시각화

###  2. 위치 수동 선택
- 사진에 GPS 정보가 없을 경우, 사용자가 **드롭다운 박스**로 지역을 직접 선택
- 추후에는 사진 분석 기반 **AI 추천 지역 기능**을 통해 정확도 향상 예정

###  3. 게시글 등록 및 상세 보기
- 게시글에 사진, 설명, 지역 정보를 함께 등록
- 게시글을 클릭하면 상세 페이지에서 확대 이미지 및 상세 설명 확인 가능

###  4. 좋아요 및 프로필 관리
- 게시글 좋아요/해제 기능
- 프로필 화면에서
  - 내 게시글 / 좋아요한 게시글 전환
  - 자기소개 수정
  - 프로필 사진 변경
  - 관심 지역 편집 가능

###  5. 지도 기반 여행 피드
- Google Maps API 또는 Kakao 지도 API를 활용
- 사용자의 여행 경로를 지도에 시각화하여 **여행 일지 형태로 표현**
